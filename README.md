# Tech-Done

A professional tech solutions website showcasing expertise, services, and portfolio.

## 🚀 Quick Start

This site is built with HTML, CSS, and JavaScript and is hosted on GitHub Pages.

### View the Site
Visit: https://js9520571-lab.github.io/Tech-done/

### Local Development
1. Clone the repository
2. Open `index.html` in your browser

## 📁 Structure

- `index.html` - Main landing page
- `css/style.css` - Styling
- `js/main.js` - Interactive features
- `assets/` - Images and media

## ✨ Features

- Professional hero section
- Services overview
- Portfolio showcase
- Contact section
- Responsive design (mobile-friendly)
- Smooth scrolling navigation

## 🔧 Technologies

- HTML5
- CSS3 (with Flexbox & Grid)
- Vanilla JavaScript
- GitHub Pages hosting

---

**Ready to customize?** Edit the content in `index.html` and update styles in `css/style.css`.
