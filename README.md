## 🚀 Getting Started

### Local Development
1. Clone the repository
   ```bash
   git clone https://github.com/js9520571-lab/Tech-done.git
   ```

2. Open in browser
   ```bash
   open index.html
   ```
   Or simply double-click `index.html`

### GitHub Pages Deployment
1. Go to repository Settings → Pages
2. Select `main` branch as source
3. Click Save
4. Your site will be live at: `https://js9520571-lab.github.io/Tech-done/`

## 🎨 Customization

### Change Colors
Edit the CSS variables in `css/style.css` (lines 11-20):
```css
:root {
    --primary-color: #667eea;      /* Change these colors */
    --secondary-color: #764ba2;
    --accent-color: #f5576c;
    /* ... */
}
```

### Edit Content
- Update company name, email, phone in `index.html`
- Modify service descriptions and names
- Change team member profiles
- Update pricing plans
- Add your portfolio projects

### Add Google Analytics
Replace `G-XXXXXXXXXX` in `index.html` with your tracking ID
